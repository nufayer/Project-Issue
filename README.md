1️⃣ What is the difference between var, let, and const?
- var has function/global scope, whereas let and const are block-scoped, they are only accessible within the nearest {}
  
2️⃣ What is the spread operator (...)?
- a JavaScript syntax that expands arrays, strings, or objects into individual elements or properties.  

3️⃣ What is the difference between map(), filter(), and forEach()?
- map() transforms each element into a new array of the same length, filter() selects a subset of elements into a new array based on a condition, and forEach() iterates over the array to perform actions

4️⃣ What is an arrow function?
- An arrow function is a compact syntax for writing functions in JavaScript

5️⃣ What are template literals?
- JavaScript string literals that allow for multi-line strings and embedded expressions using backticks (`) instead of quotes.
